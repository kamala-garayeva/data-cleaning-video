## Objective
Demonstrate a complete, step-by-step data cleaning process on a real dataset —
showing not just the code, but the reasoning and decisions behind each cleaning step.

## What the Video Covers
- **Initial data inspection** — identifying data types, structure, and immediate issues
- **Missing value detection** — locating and quantifying null/empty fields
- **Duplicate removal** — identifying and eliminating duplicate records
- **Format standardization** — fixing inconsistent date, text, and numeric formats
- **Outlier detection** — spotting anomalies that could distort analysis
- **Validation** — confirming data integrity after cleaning is complete

## Why a Video Walkthrough?
Writing clean code is one skill. Explaining *why* each decision was made is another.
This video demonstrates the ability to communicate analytical thinking clearly —
a critical skill when working with non-technical stakeholders or remote teams.

## Skills Demonstrated
- Data cleaning methodology
- Analytical communication & documentation
- Attention to data quality details
- Ability to explain technical work to any audience
